# Second project: 3d Model

## Requisites
- [Python](https://www.python.org/downloads/) 3+
- [pip](https://pip.pypa.io/en/stable/installing/)

## How to use
1. Install dependencies by entering:
```
pip install -r requirements.txt
```
2. Run the project by typing:
```
python3 project.py
```
